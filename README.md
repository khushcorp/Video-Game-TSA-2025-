# RetroBalance - Forest Level

A 2-player competitive puzzle game built with Phaser 3.

## How to Play

- **Player 1 (Solari/Gold)**: WASD controls
  - W: Jump / Latch to vine / Climb up
  - S: Force unlatch from vine
  - A/D: Move left/right

- **Player 2 (Umbrae/Purple)**: Arrow keys
  - Up Arrow: Jump / Latch to vine / Climb up
  - Down Arrow: Force unlatch from vine
  - Left/Right: Move left/right

## Objective

Compete to control puzzle nodes and influence the Balance Meter. The true goal is maintaining balance!
